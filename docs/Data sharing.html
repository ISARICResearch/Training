HOW TO SHARE YOUR DATA WITH ISARIC

STEP 1 : Signing Data Agreement

Ensure data sharing agreement is in place

STEP 2 : Sharing the Data Dictionary

Share the data dictionary of your REDCap Project.

If you used BRIDGE to build your CRF it will be really useful if you can also share with us any change in name of the variable, name on the label or choices of answers. This impact the mapping of the data to integrate them in our data platform and it will be easier with those changes are clearly identified.

How do download the Data Dictionary ?

On the menu click on Dictionary then click on Download the current Data Dictionnary

 

 

 

 

 

 

 

 

 

 

 

 


STEP 3 : Sharing the data

There are two options for sharing the data with us: either by providing access to your API or by securely sharing the data in a CSV file.

STEP 3 OPTION 1: Provide Access to Your API

We can access your data directly through the REDCap API. To do this, you will need to:

1.       Add the ISARIC data manager as a user on your REDCap instance.

2.       Grant them access to your database.

3.       Generate an API token for them to enable access.

Detailed instructions are provided below.:

1.       Add the ISARIC data manager as a new user on your REDCap instance.

If you cannot access the Control Center it does not appear, it means that you do not have administrator rights, please contact the administrator to add the new user.
 
Click on Control Center

 


 

 

 

Click on Add Users (Table-based Only)

 

Complete the information and click on Save at the bottom of the page and the user account will be created. 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

2.        Grant them access to your database.

Enter the user name of the ISARIC Data Manager and click on +Add with custom Rights

 
 
Click on User Rights

 

 

 

 

 

 

 

 

 

 

 


And complete the privileges as follows :

Click on Add user .

The ISARIC Data manager will be notified that they have now access to the database. They will access to the REDCap project and request for an API token and send you a message to inform you the request has been made.

3.       Generate an API token for them to enable access.

 

 The API token request from the ISARIC Data manager will appear in the Control Center To-Do List . If you do not have Administrator rights please ask you administrator to approve this request.

Click in To-Do List

 

Click on the Green tick to approve the request.

A pop window appears, click on Create a new API token then Close

The ISARIC Data manager can now export directly from your database the data.

STEP 3- OPTION 2 : Sharing securely a CSV file

1.       Download the data in CSV

 

 

2.       Send securely the data

Do not send your data by email, it is not secure. REDCap has an integrated secure tool for sending data.

Click on the Welcome page of REDCap on the Send-It tools

 

Complete the address of the ISARIC Data Manager, email subject and select your csv file and click on Send it

If your file is bigger than the max file size of 128MB, please contact us at data@isaric.org
